# PHP Login & Dashboard Project

A simple PHP login system with user authentication, profile management, and a basic dashboard. Built using **PHP**, **MySQL**, and **Bootstrap 5**.

## Features

- User Registration & Login
- User Profile Management (Update Profile, Profile Image)
- User Role Support (Admin/User)
- Dark Mode Toggle
- Switch Between Users (for testing)
- Secure Session Handling
- Bootstrap 5 Responsive UI

## Setup Instructions

1. **Clone the repository:**
```bash
git clone https://github.com/paulexmaze99/login.git
cd login
Setup XAMPP/LAMP/WAMP:

Place the login folder in htdocs (XAMPP) or www (WAMP/LAMP).

Database Setup:

Create a MySQL database (e.g., login_system).

Import the database schema from mysql database table.txt.

Update config.php with your database credentials.

Run the Project:

Start Apache and MySQL.

Open your browser and go to:
http://localhost/login/login.php
